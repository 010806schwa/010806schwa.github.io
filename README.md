# Personal Homepage

这是林昊天的 GitHub Pages 静态个人主页，风格参考学术个人主页：左侧个人信息栏，右侧展示简介、动态、论文、项目、经历和荣誉。

## 修改内容

- 在 `index.html` 中继续完善 GitHub、Google Scholar、论文和项目链接。
- 将 `assets/avatar.svg` 替换成你的个人照片，文件名保持为 `avatar.svg`，或同步修改 `index.html` 里的图片路径。
- 最新简历下载文件位于 `assets/resume-haotian-lin-20260412.docx`。

## 发布到 GitHub Pages

1. 将这些文件推送到你的 `username.github.io` 仓库。
2. 在 GitHub 仓库的 Settings > Pages 中选择主分支发布。
3. 稍等片刻后访问 `https://username.github.io`。
